## Description

Add a 'draft' label for epics that makes them flow through the kanban like normal issues. Draft epics appear in the kanban board columns alongside tasks/bugs/features. An epic_planner focus picks up draft epics and breaks them into child tasks with dependencies. When planning is complete, the agent removes the draft label and sets the epic to deferred, at which point it becomes a normal epic container visible only in swimlane view. The swimlane view gets a toggle to add/remove the draft label.
